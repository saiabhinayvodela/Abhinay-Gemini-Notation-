# Abhinay-Gemini-Notation-
A mathematically complete, scalable, and algebraic extension for Roman numerals co-created by Vodela Sai Abhinay and Gemini.


---

## 🏛️ The Three Laws of AG Notation

### 1. The Negative Operator (`N`)
The Latin letter **N** (*nulla/nihil*) placed at the absolute front of a Roman numeral block acts as a negative modifier. 
* *Examples:* 
  * `NX` = -10
  * `NCD` = -400

### 2. The Absolute Zero
When the letter **N** stands alone without any trailing numerals, it represents the absolute value of zero.
* *Example Equation:* `X + NV = V` (Translates to: 10 + (-5) = 5)
* *Example Equation:* `X + NX = N` (Translates to: 10 + (-10) = 0)

### 3. The Multiplier Power (Superscript)
Massive scales are handled using superscript multipliers instead of traditional messy stacked bars, multiplying the base by powers of 1,000.
* *Example:* `M^IV` = 1,000 to the 4th power = 1 Quadrillion
